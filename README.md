# My First Website (Web Developer Portfolio)

**Live Demo:** [https://ssiyam0123.github.io/my-1st-website/](https://ssiyam0123.github.io/my-1st-website/)

A clean, modern, and professional web developer portfolio website showcasing skills, education, experience, and contact options.

## Features
- **Hero Banner**: Highlighting introduction, summary, CV download, and contact triggers.
- **About Me**: Brief bio with basic details (Email, Date of Birth, Location).
- **Skills Section**: Grid overview of technical skills like JavaScript, React, Node.js, and MongoDB.
- **Resume Section**: A clean timeline showing educational background and professional experience.
- **Contact Form**: Responsive layout allowing users to send messages and find social profiles.

## Technologies Used
- HTML5
- CSS3 (Vanilla CSS, Fully Mobile Responsive)
- Google Fonts (Open Sans)
- FontAwesome (for icons)

## Setup & Running Locally
1. Clone this repository.
2. Open `index.html` in your favorite web browser.
3. (Optional) Run using a local development server like Live Server in VS Code.
